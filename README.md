## Getting Started

Install the module with: `npm install sg-read`

```js
var sg-read = require('sg-read');
sg-read.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g sg-read
$ sg-read --help
$ sg-read --version
```


```sh
# creates a browser.js
$ grunt browserify
```
